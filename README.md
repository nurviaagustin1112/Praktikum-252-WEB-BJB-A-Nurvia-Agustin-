# Praktikum 252 - WEB BJB A

Aplikasi Sistem Informasi Inventaris Barang berbasis Laravel 11 yang dibuat untuk memenuhi tugas Praktikum Web BJB A.

## Fitur

- Login dan Logout (Authentication)
- Dashboard
- CRUD Barang
- CRUD Barang Masuk
- CRUD Kategori
- CRUD Pemasok
- CRUD Karyawan
- CRUD Ruang

## Perubahan yang Ditambahkan

### 1. Fitur Login
- Menambahkan autentikasi pengguna menggunakan Laravel Authentication.
- Pengguna wajib login sebelum mengakses dashboard dan seluruh menu aplikasi.
- Menambahkan fitur logout dengan notifikasi **"Anda berhasil keluar."**.

### 2. Penambahan Tabel Kategori
Menambahkan tabel **kategori** untuk mengelompokkan data barang sehingga data inventaris menjadi lebih terstruktur.

Struktur tabel:
- id
- nama_kategori
- keterangan

Contoh kategori:
- Elektronik
- Komputer
- Furniture
- Alat Tulis
- Jaringan
